NUS新生入学指南——行李篇

1. 前言

打算将这个博客经营为一个新加坡cs留子的生存指南。正好马上就要启程去新加坡了，马上要开始收拾行李，也顺手记录一下需要带的东西，充当一下备忘录。

预计本篇文章至少需要两次完成，一次是在国内预计我需要带什么，肯定准备的不会很完善的，所以到了新加坡我还会更新一次落了哪些东西。

2. 准备

在国内想到的必带物品

2.1 材料

必备材料：

  • 身份证：原件 * 1 + 复印件 * n ✅
  • 护照： 原件 * 1 + 复印件 * n ✅
  • 证件照：一寸、两寸、小二寸 ✅
  • 雅思成绩单 ✅

Registration Part 2:

  • 学位证书、毕业证书、成绩单：中英文原件 * 1 + 中英文盖章复印件 * n ✅
  • 盖本科院校骑缝章的信封 ❓

OSE：

  • 新加坡手机号
  • IPA Letter：在右上角写上新加坡本地号码 ✅
  • 两寸绒面白底证件照 * n
  • ICA两次付款收据：Student Pass Issuance Fee ($60) + Mutiple Journey Visa ($30)
  • 体检报告：NUS体检表 * 1 + ICA体检表 * 1 + HIV报告 * 1 + 胸片报告 * 1 ✅
  • 20新：如果证件照不合规会现场照 ✅
  • e-visit Pass（目前不知道是啥）
  • 短期访问签（STVP，目前也不知道是啥）

2.2 床具

  • 褥子 * 1：1.2m * 2m，对应新加坡的super single size（1.1 * 1.9） ✅
  • 枕头 * 1 ✅
  • 枕套 * 2 ✅
  • 枕巾 * 2 ✅
  • 毛巾被 * 1 ✅
  • 床单 * 2 ✅

2.3 衣服

  • 休闲：
    • 短袖 * 8 ✅
    • 短裤 * 3 ✅
    • 长袖外套 * 2 ✅
    • 牛仔裤 * 3 ✅
  • 运动：
    • 速干短袖 * 3 ✅
    • 速干短裤 * 2 ✅
  • 防晒服
  • 鞋：
    • 休闲鞋 * 2 ✅
    • 篮球鞋 * 1 ✅
    • 跑步鞋 * 1 ✅
    • 拖鞋 * 1 ✅
  • 内裤 * 6 ✅
  • 袜子 * 10 ✅
  • 买一些一次性内裤 ✅

2.4 生活用品

日常：

  • 眼罩、耳塞：在京东多买几个 ✅
  • 唇膏 * n ✅
  • 花露水 * 1、蚊子药 * 1 ✅ （花露水上不了飞机）
  • 蟑螂药 ✅
  • 一次性手套 ✅
  • 水银温度计 * 1 ✅
  • 塑料袋 * n ✅
  • 纸：抽纸、卷纸、小包装纸、湿纸巾 ✅
  • 压缩毛巾（抹布） ✅
  • 盆 * 3 ✅
  • 筷子 ✅

洗浴：

  • 牙膏 * 2、牙刷 * n ✅
  • 香皂 * 2、肥皂 * 2 ✅
  • 洗发水 * 1 ✅
  • 洗面奶 ✅
  • 吹风机 ✅
  • 刮胡刀 ✅
  • 毛巾 ✅
  • 浴巾 ✅

2.5 电子产品

  • 电脑、手机、平板、耳机
  • 充电线：type c（手机、iPad、耳机）* 2 + 电脑充电线 ✅
  • 移动硬盘 ✅
  • 拓展坞、use-typec转换头 ✅
  • 充电转换头 * 2 ✅
  • 电话卡针 ✅
  • 插排 * 2 ✅

2.6 药

  • 感冒药（清热颗粒、口服液、白加黑） ✅
  • 红霉素眼膏 ✅
  • 肠胃药（吉法酯） ✅
  • 皮炎平 ✅
  • 创可贴 ✅
  • 酒精碘伏 ✅
  • 维生素B ✅

3. 补充
